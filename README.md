# Cora's Social Spark App 

## Description
This Android application was developed using Kotlin in Android Studio.  
The purpose of the app is to help users maintain social connections by suggesting small social actions ("sparks") based on the time of day entered by the user.

---

## Features
- User input field to enter time of day (e.g., morning, afternoon, dinner)
- Displays a social suggestion based on input
- Reset button to clear input and results
- Error handling for invalid inputs

---

## Social Spark Suggestions
- **Morning:** Send a "Good morning" message to a family member  
- **Mid-morning:** Send a quick "Thank you" message to a colleague  
- **Afternoon:** Share a meme or interesting link with a friend  
- **Snack time:** Send a "thinking of you" message  
- **Dinner:** Call a friend or relative
